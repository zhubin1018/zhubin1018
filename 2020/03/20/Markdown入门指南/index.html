使用 iOS 版本印象笔记如何快速保存内容？

  1. 启用印象笔记 Widget ——印象笔记·剪贴板
  2. 复制粘贴任意内容
    • 微信
  3. 滑动到 Widget 插件区域即可完成保存
    印象笔记·剪贴板有什么特点？
    * 快：开启自动模式，可以自动保存剪贴板的任意内容
    * 一切：只要可以复制粘贴就可以保存
    * 有序：全部保存在「我的剪贴板」笔记本并以时间来命名
